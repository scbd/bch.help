Программный модуль «AJAX» для разработки национального МПБ


Программный модуль МПБ «Ajax Plug-In» – это простое JavaScript приложение, основанное на технологии Ajax, которое может быть легко интегрировано в любой веб-сайт для отображения в динамическом режиме информации Центрального портала МПБ.

Использование программного модуля МПБ «Ajax Plug-In» не требует специфических знаний в области информационных технологий. Он работает на любых серверных платформах и не требует установки дополнительного программного обеспечения. Все необходимые файлы находятся на Центральном портале МПБ и поддерживаются Секретариатом КБР.

Программный модуль МПБ «Ajax Plug-In» является «клиентским» приложением. Соединение с Центральным порталом МПБ осуществляется непосредственно клиентом (непосредственным пользователем).
