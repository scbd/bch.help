Приложение «Гермес» для разработки национального МПБ
Оперативная помощь Механизма посредничества по биобезопасности > Обучающие материалы > Приложение «Гермес» для разработки национального МПБ

Glossary Item Box

Секретариат Конвенции о биологическом разнообразии (СКБР) разработал приложение «Гермес», которое может быть использовано странами, которые хотят иметь свои веб-сайты МПБ, но не имеют для этого достаточно ресурсов или не желают разрабатывать собственные системы.

Приложение «Гермес» работает на серверах СКБР и обрабатывает данные, относящиеся к конкретной стране. Странам необходимо иметь только компьютеры с браузером и доступом в Интернет. Настройка приложения также осуществляется через Интернет.
