Glossary Item Box

The BCH Modules comprise a set of cohesive training subunits. The training subunits are self-contained, which means that different components can be put together in various formats suitable for different target audiences.

Note:

  • You will need to uncompress the file and to write the ISO image to a CD-ROM. For instructions on how to do this, consult the manual provided with your CD-ROM writing software.
  • ISO images are not saved to CD-ROMs in the same way as regular data files.
