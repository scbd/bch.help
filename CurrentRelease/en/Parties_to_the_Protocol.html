Glossary Item Box

The link for Parties to the Protocol takes the user to a page which provides a summary of the status of ratification and entry into force for each Party to the Protocol. The table lists countries that have signed, ratified, accepted, approved or acceded to the Protocol and the date at which the Protocol entered into force in each country. Each country name provides a link to the Country Profile, which provides summary information about all records registered in the BCH by that country.
