Glossary Item Box

The Online Forums and Portals page provides numerous links to forums and portals that can help the BCH user to find further information. This page is organized into several sections:

 

  • Forum News section, displayed at the top of the page, shows a list of latest news from the forums.
  • The Ongoing Forums section provides a list of links to currently active forums.
  • The Archived Forums section provides a list of links to closed forums.
  • The Archived Surveys section provides access to a list of surveys previously conducted.
  • The Collaborative Portals section provides links to several portals related to biosafety, in which the user can find further information and assistance.
